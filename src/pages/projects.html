Ongoing projects

NAS Infrastructure

A self-hosted, redundant server built to replace commercial cloud services for media and backups. It utilizes Linux (Open Media Vault), Docker, and RAID 1 to manage media streaming, backups, and private game servers locally.

Linux Docker RAID 0/1

IoT Weather Analysis

An engineering project to build a low-cost, local weather station using an ESP32 microcontroller. The goal is to collect environmental data and use Python to perform a comparative accuracy study against major algorithmic forecasts.

Python Microcontroller Data Analysis

Ethical Supply Chain Audit

A quantitative analysis project mapping the relationships between consumer products and their parent entities. Uses graph databases to trace supply chains and calculate the "convenience tax" of certified ethical alternatives, visualizing the impact of supply chain sovereignty.

B-Corps Data Science Supply Chain Analysis Neo4j Graphing

PoE Privacy-First Video Doorbell

An offline doorbell connected to my NAS via ethernet, accessible through the Home Assistant app running on my NAS. This project was designed as an alternative to other video doorbells, where all data is stored locally and nothing is ever sent to the cloud, safeguarding privacy and reducing the risk of data being mishandled.

Privacy Anti-surveillance Linux Home Assistant
